1. 
git init : để khởi tạo repo (working directory)
git add <file_name>: thêm file vào vùng chờ commit (staging area)
git add .: thêm tất cả các file vào vùng chờ commit (staging area)
git commit -m "message": commit code thông điệp là (message)
2. 
câu chạy lệnh 2c bị lỗi do chưa add vào vùng staging area đã commit 
câu chạy lệnh 2f bị lỗi do lệnh đó chưa có gì ở vùng staging area để commit
